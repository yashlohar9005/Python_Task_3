Grade Calculator (Python) 

Author:- Yash Lohar

A simple Python program to calculate student grades using conditional statements, logical operators, and nested decision logic.

This project is part of TASK 3: Conditional Statements & Logical Flow and demonstrates how to write decision-based business logic in Python.

_________________________________________________________________________

Tools Used

  • Python

  • VS Code

_________________________________________________________________________


Features

• Takes marks input from user

• Validates input range (0–100)

• Uses if-elif-else

• Uses logical operators (and, or)

• Uses nested conditions

• Displays proper grade messages

• Handles invalid input using try-except


_________________________________________________________________________


Grade Rules

90 – 100 : A+ (Excellent)

80 – 89  : A (Very Good)

70 – 79  : B (Good)

60 – 69  : C (Average)

50 – 59  : D (Pass)

Below 50 : F (Fail)
